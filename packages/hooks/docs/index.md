## 快速开始

使用@tiny/hooks 的基本指南。

### 设置 npm 仓库

```
npm config set registry http://artifactory.sf-express.com/artifactory/api/npm/npm/
```

### 安装 @tiny/hooks

```
npm install @tiny/hooks
```

or

```
yarn add @tiny/hooks
```
